# hello-world-everywhere
### **hello-world-everywhere** is a project to have an example "Hello, World!" in every language available.

Current Supported Languages:
* BASIC
* Bash
* Batch
* C
* C#
* C++
* CSS
* D
* F#
* Golang
* Haskell
* HTML
* Java
* JavaScript
* Lua
* Objective-C
* Perl
* php
* PowerShell
* Python
* Ruby
* Ruota 
* Swift
* Visual Basic
* Visual Basic Script
